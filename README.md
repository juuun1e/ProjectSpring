- FrontEnd : (React / JavaScript / VSCode)
- BackEnd : (Spring Boot / Java / STS), (Django / Python / PyCharm)

## 📍Topic
- `충전소 위치`와 배치되어 있는 `충전타입 종류`,  `실시간 정보`,  `주차장 요금 정보`제공
- 사용자 간 정보 공유를 할 수 있는 커뮤니티 장 제공
- 인 차량 충전 타입에 맞는 충전소 위치 검색 기능
